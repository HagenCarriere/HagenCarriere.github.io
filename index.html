Projects

  • YouMadeIt Engine
    Description: C++ game engine with lua as a guest language.
    Contributions: I developed the engine and used technologies such as SDL for window management, Box2D for physics, LuaBridge for lua integration, and Tiled for level design.
    Technologies Used: C++, Lua, Tiled, GLM, LuaBridge, SDL, Box2D
  • Card King
    Description: Deckbuilding adventure game, with a top down view. Battles are in real time with a recharging mana bar. Solo adventure with a side PvP mode.
    Contributions: I developed the Cutscenes throughout the game as well as the scripted fights such as the tutorial and final boss.
    Technologies Used: C# and Unity
  • p1-Zelda
    Description: Recreation of the first dungeon from the original Legend of Zelda game in Unity. Pressing 4 gives access to a custom dungeon.
    Contributions: I developed the core weapon system, Link sprite behavior, the secret rooms, and the custom weapon and it's mechanics.
    Technologies Used: C# and Unity
  • Sliding Stars
    Description: A platforming game with a unique mechanic where the player can spawn a clone and swap loactions with it.
    Contributions: I developed the game including: core mechanic, the level design, and the player sprite behavior.
    Technologies Used: C# and Unity
